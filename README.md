This is the github repository for the ICML'26 paper entitled

[Incorporating Importance Weighting in Optimal Transport based Domain Alignment](https://openreview.net/forum?id=R8HbSt2J4e).

Our contributions in a nutshell: We introduce importance weighting to optimal transport based domain adaptation bounds and show the connection to unbalanced optimal transport.

You can find the poster for the paper in the poster/ folder. Email okan.koc@riken.jp if you have any questions.
