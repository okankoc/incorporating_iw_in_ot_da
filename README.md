This is the github repository for the ICML'26 paper entitled

[Incorporating Importance Weighting in Optimal Transport based Domain Alignment](https://openreview.net/forum?id=R8HbSt2J4e).

#### Our contributions in a nutshell

We introduce importance weighting to optimal transport based domain adaptation bounds and show the connection to unbalanced optimal transport.

#### Code

To run the code locally please call `python run_uda.py --config local`. Change the argument to `cluster` to use the cluster config. You can access and edit the config files
in the `config/` folder.
